# AI Research Assistant 🤖

An AI-powered research assistant that lets you upload PDF papers and ask questions about them using RAG (Retrieval Augmented Generation).

##  Live Demo
https://ai-research-assistant-snowy-beta.vercel.app

##  Features
- Upload any PDF research paper
- Ask questions in natural language
- Get AI-powered answers based on the paper
- View source citations for every answer
- Built with a full RAG pipeline

## Tech Stack
| Part | Technology |
|------|-----------|
| Frontend | React |
| Backend | FastAPI |
| AI Framework | LangChain |
| Embeddings | OpenAI text-embedding-3-small |
| LLM | GPT-4o-mini |
| Vector DB | ChromaDB |
| Frontend Deploy | Vercel |
| Backend Deploy | Render |
